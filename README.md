# GitHub-OpenList-Sync

*Powered by AI*

**自动同步GitHub仓库到OpenList网盘**

*虽然这很神经，但真的很好玩*

## 用法

```bash
git clone https://github.com/TGBUG/GitHub-OpenList-Sync
cd GitHub-OpenList-Sync
cp example_config.yaml config.yaml
```

然后修改config.yaml

**启动**

```bash
python main.py
```

**若要单次运行，不启动管理面板**

```bash
python main.py --once
```
